# ReCreate Living

A premium handcrafted sofa business website with a full Admin Dashboard.

## 🚀 Getting Started

### Backend (Port 3001)
```bash
cd server
npm install
node index.js
```

### Frontend (Port 5174)
```bash
cd client
npm install
npm run dev
```

## 🔑 Admin Access
- **URL:** http://localhost:5174/admin
- **Email:** admin@recreateliving.com
- **Password:** Admin@123

## ✨ Features
- Public storefront with live product listings
- Admin login with role-based access
- Product management (Add / Edit / Delete)
- Stock management (increment/decrement per product)
- Team management (add team members with roles)
